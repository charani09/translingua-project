TransLingua is a simple AI-powered multi-language translator built using
Streamlit and Google Gemini Generative AI.

It allows users to translate text from one language to another using a
clean and interactive web interface.

## Features
- Text-based language translation
- Supports multiple languages
- Simple Streamlit UI
- Uses Gemini Flash model

## Technologies Used
- Python
- Streamlit
- Google Generative AI (Gemini)
- dotenv

## How to Run the Project

1. Install required libraries:
   pip install -r requirements.txt

2. Create a `.env` file and add your API key:
   GOOGLE_API_KEY=your_api_key_here

3. Run the application:
   streamlit run translang.py

## Output
The application translates the given text from the selected source
language to the target language and displays the result on the screen.
